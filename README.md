### Getting Started:

#### Get the Data:
1. Get the data csv from
[kaggel.com](https://www.kaggle.com/sudalairajkumar/novel-corona-virus-2019-dataset)

2. Add it to the data folder

#### Install Dep:
1. Make a venv:
    python -m venv venv

2. Activate it:
    .\venv\Scripts\Activate.ps1
    
3. Install Req:
    pip install -r requirements.txt
